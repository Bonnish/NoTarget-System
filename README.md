# 🛠️ No Target System (Bonnish Utilities)

A clean and efficient Toolgun-based utility for Garry's Mod that manages the no target state. This tool simplifies NPC interaction management for administrators and developers.

## 🚀 Features
- **Target Toggle:** Switch NoTarget status on any player you are looking at.
- **Self Toggle:** Quickly change your own visibility to NPCs.
- **In-Chat Feedback:** Real-time notifications indicating the new state (ENABLED/DISABLED) and the player's name.

## 🎮 How to use
Find the tool in the **"Bonnish Utilities"** category within your Spawn Menu:

* **Left Click:** Toggle NoTarget on the player you are looking at.
* **Right Click:** Toggle NoTarget on yourself.

## 📂 Installation

1. Download the repository as a ZIP or clone it using Git.
2. Place the folder into your `garrysmod/addons/` directory.
3. Restart your game or server.
