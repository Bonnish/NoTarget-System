# Bonnish Utilities - No Target System

**No Target System** is a robust and highly configurable Garry's Mod addon that allows administrators or designated roles to seamlessly toggle "NoTarget" mode. While in NoTarget mode, NPCs will completely ignore the player, making it an essential tool for admins managing events or building without interruptions.

This addon is fully integrated with the **Bonnish Utilities Core**, allowing you to configure all settings directly through a beautiful in-game UI.

## ✨ Key Features

- 👻 **NPC Invisibility:** Toggle NoTarget mode to become completely invisible to hostile and friendly NPCs.
- ⚙️ **In-Game Configuration:** Configure permissions, chat commands, and features directly via the Context Menu (Hold C) dashboard.
- 💬 **Customizable Chat Command:** Choose your preferred chat command (defaults to `!notarget`).
- 🔫 **Admin Toolgun Tool:** Includes a custom STool to shoot other players and instantly toggle their NoTarget status (if allowed by settings).
- 👔 **DarkRP Job Auto-Toggle:** Easily select which DarkRP jobs should automatically spawn with NoTarget enabled (e.g., Admins on Duty).
- 🌍 **Multi-Language Support:** Fully translated into English and Spanish via the Bonnish Core.
- 🔔 **Clean Notifications:** Aesthetic on-screen notifications instead of chat spam.

## 📥 Installation

1. Download this repository as a `.zip`.
2. Extract and rename the folder to `NoTarget-System`.
3. Drop it into your `garrysmod/addons/` folder.

> ⚠️ **REQUIREMENT:** This addon depends entirely on the [**Bonnish Utilities Core**](https://github.com/Bonnish/bonnish-utilities-core). Without the Core installed on your server, this addon will not function or appear in your menus.

## 🛠️ Configuration

You do not need to edit any Lua files to configure this addon. Simply join your server, open the Context Menu (hold `C`), click on the Bonnish icon, and navigate to the **No Target System** module.

Available in-game settings:
- **Allow Self No Target:** Allow players with permissions to use the command on themselves.
- **Allow Target Others:** Allow admins to use the command or Toolgun to grant NoTarget to other players.
- **Chat Command:** Change the trigger command (e.g., `!stealth`, `!notarget`).
- **Allowed Jobs (DarkRP):** A visual multi-select list of jobs that will always spawn with NoTarget active.
